About The Website

100% handedcoded by human

the entire site is coded using notepad++. layout graphic was sketched, finalized, and dismembered on gimp.

technical information

  1. hosted on github pages. all opinions represented across this URL are my own and do not reflect on any of the third parties used to make this site.
  2. minimal javascript. content is formatted in basic markdown and should be screenreader-accessible. this site is completely static with zero client-side so the only data storage occurring cycles within the site alone. these scripts only stitch visual assets together into a single uniform using doc.write and raw data.
  3. imagemaps and some screenreader-unfriendly elements are present.
